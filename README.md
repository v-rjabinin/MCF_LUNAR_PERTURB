# MCF_MOON_PERTURB

Данная лабораторная работа посвящена изучению влияния гравитационного поля Луны на движение искусственного спутника Земли (ИСЗ) методом вариации произвольных постоянных. В работе выполняется численное интегрирование системы дифференциальных уравнений, расчёт возмущающих ускорений и построение графиков эволюции оскулирующих элементов орбиты.

## ⚙️ Что реализовано

1) Математическая модель возмущённого движения

    * Система дифференциальных уравнений в оскулирующих элементах
    * Уравнения для $p$, $e$, $i$, $Ω$, $ω$
    * Дополнительное уравнение для вычисления производной времени $\frac{dt}{du}$

2) Преобразование систем координат

    * Переход из инерциальной системы (ECI/АГЭСК) в геоцентрическую орбитальную систему (ГОСК) с помощью матриц поворота
  
3) Вычисление возмущающих лунных ускорений

    * Радиус-векторы «Луна-Земля» и «Луна-ИСЗ»
    * Проекции ускорения в ГОСК ($F_x$, $F_y$, $F_z$)
    * Орбитальные компоненты ($S$, $T$, $W$)

4) Численное интегрирование

    * Метод `RK45` (Рунге-Кутты 4(5) порядка)
    * Использование функции `solve_ivp` из $SciPy$

5) Расчёт положения Луны

    * Синхронизация времени для спутника и Луны
    * Решение уравнения Кеплера итерационным методом
    * Вычисление эксцентрической и истинной аномалий

6) Валидация данных

    * Дескрипторы и properties для проверки корректности
    * Проверка диапазонов орбитальных элементов
    * Обработка краевых ситуаций

<a id="-основные-задачи-проекта"></a>
## 📈 Основные задачи проекта

1) Моделирование лунных возмущений

    * Учёт гравитационного воздействия Луны на ИСЗ
    * Вычисление возмущающего ускорения по формулам задачи трёх тел

2) Анализ эволюции орбиты

    * Исследование изменения $p$, $e$, $i$, $Ω$, $ω$ на различных интервалах
    * Выделение периодической и вековой составляющих возмущений
    * Построение графиков для $u ∈ [0.18π; 10π], [0.18π; 30π], [0.18π; 73.6π]$

3) Изучение компонент ускорения

    * Построение графиков радиальной, трансверсальной и бинормальной компонент
    * Анализ зависимости от расстояния между КА и Луной
    * Определение точек экстремумов и других особенностей

4) Синхронизация движения

    * Корректный расчёт времени для каждого положения спутника
    * Определение положения Луны в соответствующий момент времени

5) Практическая значимость

    * Оценка влияния лунных возмущений на высокоэллиптические орбиты
    * Выводы о необходимости учёта возмущений для точного прогнозирования
    * Анализ поворота плоскости орбиты и вращения линии апсид

## ⚙️ Установка

1. **Клонируйте репозиторий**:
```bash
git clone https://github.com/v-rjabinin/MCF_LUNAR_PERTURB.git
cd MCF_LUNAR_PERTURB
```
2. **Установите зависимости**:
```bash
pip install numpy scipy matplotlib
```
3. **Запустите программу**:
```bash
python3 main.py
```
4. **Просмотр результатов**:

* Графики откроются в окнах `Matplotlib`
* Подробный анализ содержится в `Report.pdf`
* Исходные данные можно изменить в `data.py`
